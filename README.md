# QurbaniHat

## Project Purpose

QurbaniHat is a modern livestock marketplace web application built with Next.js.  
It allows users to browse animals for Qurbani, view detailed information, and book animals after login. It also includes authentication (email/password + Google login) and user profile management.

---

## 🌐 Live URL

https://qurbani-hat-mocha.vercel.app/

---

## ✨ Key Features

- 🔐 User Authentication (Email/Password + Google Login using Better Auth)
- 👤 User Profile (Name, Email, Profile Image display)
- 🐄 Animal Listing Page (All Animals)
- 📄 Animal Details Page (Dynamic route `/AllAnimals/[id]`)
- 📦 Booking System (Protected route - login required)
- 🧑‍💼 Profile Update Page (Name & Image update feature)
- 🔒 Protected Routes (Redirect to login if not authenticated)
- 🧭 Dynamic Navbar (Shows user info or login/register buttons)
- 🔔 Toast Notifications (Success & Error handling)
- 📱 Fully Responsive UI (Tailwind CSS)
- ⚡ Fast performance using Next.js App Router
